## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/playstore-review-analyzer.git
    cd playstore-review-analyzer
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3.  Set up your Google Gemini API Key:
    *   Get a key from [Google AI Studio](https://aistudio.google.com/).
    *   Set it as an environment variable in your terminal:
        ```bash
        export GEMINI_API_KEY="your_actual_api_key_here"
        ```

## Usage

Run the main scraper script:

```bash
python review_scraper.py
```

Follow the interactive menu:
1.  **Fetch New Reviews**: Search for an app and download reviews.
2.  **Analyze Existing CSV**: Run AI analysis on previously fetched data.
3.  **Fetch & Analyze**: Run the end-to-end pipeline.

## Output

Results are saved in the `outputs/` directory:
*   `{app_id}_reviews.csv`: Raw review data.
*   `{app_id}_reviews_analyzed_ai.csv`: AI-tagged dataset with Category and Priority.
*   `{app_id}_roadmap.md`: Strategic roadmap document generated by the AI CPO.
